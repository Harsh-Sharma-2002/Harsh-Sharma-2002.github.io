Clinical Specialty Classification

RoBERTa Embeddings + Random Forest for Medical Text Classification

View Repository

Overview

This project focuses on automatic classification of medical transcriptions into clinical specialties such as cardiology, radiology, orthopedics, and surgery. Instead of training a deep neural classifier end-to-end, the system uses transformer-based embeddings (RoBERTa) combined with a classical Random Forest classifier for efficient and interpretable prediction.

Problem Statement

Healthcare systems generate large volumes of unstructured clinical notes. Efficiently categorizing these notes by specialty is essential for:

  • Electronic Health Record (EHR) indexing
  • Automated document routing
  • Faster insurance claim processing
  • Downstream NLP tasks (NER, ICD coding, summarization)

System Architecture

  1. Select top-k most frequent specialties
  2. Generate RoBERTa embeddings (mean pooled)
  3. Encode labels using LabelEncoder
  4. Apply Random Oversampling for class balance
  5. Train RandomForest with GridSearchCV
  6. Evaluate with classification report & confusion matrix

Model Design Decisions

  • Used roberta-base for semantic embeddings
  • Mean pooling across token representations
  • Random Forest for robust non-linear classification
  • GridSearchCV for hyperparameter tuning
  • RandomOverSampler to address class imbalance

Results

The final model achieved:

  • ~86% overall accuracy on the test set
  • Balanced precision and recall across top specialties
  • Improved minority-class recall after oversampling

The confusion matrix and classification report are generated and saved as part of the training pipeline.

Why This Project Matters

This project demonstrates practical NLP system design: combining transformer embeddings with classical ML models for efficient training and strong performance without requiring full fine-tuning of large language models.

It highlights the tradeoff between deep learning complexity and operational simplicity — showing that hybrid approaches can achieve competitive performance while remaining computationally efficient.
